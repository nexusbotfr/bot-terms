# Politique de Confidentialité - NexusBot

**Dernière mise à jour : 13 février 2026**

Cette Politique de Confidentialité explique comment **NexusBot** (ci-après « le Bot ») collecte, utilise et protège les informations lorsque vous l'utilisez sur Discord.

## 1. Données collectées
Le Bot collecte uniquement les données strictement nécessaires à son fonctionnement, notamment :
- Identifiants Discord (ID utilisateur, ID serveur, ID message, ID salon)
- Contenu des messages (uniquement pour les commandes, logs, tickets, starboard, notifications RSS)
- Configurations du serveur (rôles, salons, flux RSS, etc.)
- Données de modération (warns, mutes temporaires, bans temporaires)

Aucune donnée personnelle sensible (mot de passe, email, numéro de téléphone, etc.) n'est collectée.

## 2. Utilisation des données
Ces données sont utilisées exclusivement pour :
- Exécuter les commandes du Bot
- Stocker les configurations par serveur
- Fournir les fonctionnalités (tickets, starboard, RSS, modération, etc.)
- Générer des logs internes (pour debug et support)

Aucune donnée n'est vendue, louée ou partagée avec des tiers à des fins commerciales.

## 3. Stockage et sécurité
Les données sont stockées sur des serveurs sécurisés (hébergement privé ou cloud avec chiffrement).  
Nous prenons des mesures raisonnables pour protéger ces données, mais aucun système n'est infaillible.

## 4. Suppression des données
Vous pouvez demander la suppression des données liées à votre compte ou serveur :
- Pour un serveur spécifique : retirez le Bot du serveur (toutes les configs seront effacées)
- Pour vos données personnelles : contactez-nous en privé

Nous supprimerons les données dans un délai raisonnable après demande valide.
